Anthropic Engineering · Engineering Blog

Managed Agents 架构：把 brain 与 hands 解耦

Anthropic 介绍 Claude Managed Agents 的工程设计：将 agent 分成 session、harness 和 sandbox 三个可替换接口，避免把所有组件塞进单一 container 形成难以恢复的“pet”。新架构让 harness、sandbox 都可像 cattle 一样失败后重建，session log 独立持久化；同时把凭证隔离在 sandbox 外，并用 getEvents() 让 session 成为 Claude context window 之外的可查询上下文对象。

Claude Blog · Product Blog

Claude Managed Agents 支持 self-hosted sandboxes 与 MCP tunnels

Claude Managed Agents 新增 self-hosted sandboxes 和 MCP tunnels。企业可把 agent 的工具执行放在自有基础设施或 Cloudflare、Daytona、Modal、Vercel 等 managed providers 中，agent loop 仍由 Anthropic 处理；MCP tunnels 则让 agent 通过出站连接访问私有网络中的 MCP servers，无需公开 endpoint。self-hosted sandboxes 已 public beta，MCP tunnels 为 research preview。
