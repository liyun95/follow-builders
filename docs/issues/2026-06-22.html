Claude Managed Agents：将 brain 与 hands 解耦

Anthropic 介绍 Managed Agents 的系统设计：把 session、harness、sandbox 虚拟化为可替换接口。核心思路是将“brain”（Claude 与 harness）从“hands”（sandbox 和工具）以及持久 session log 中解耦，使容器和 harness 都可以像 cattle 一样失败、重启与替换，同时降低凭证暴露风险并支持长时程任务的上下文恢复。
