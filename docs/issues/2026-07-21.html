用一个 agent 做事，另一个 agent 按 rubric 审核

Peter Yang 引用 Thariq 的观点：在判断视频 short 是否“好”这类非确定性任务中，应让独立 verification agent 读取 rubric、审查结果并给出反馈。原因是 self-preferential bias：模型在验证自己的输出时往往更宽松，因此生产与审核最好拆成两个 agent。

Aaron Levie · CEO @box

多模型 agentic systems：用 frontier planner 搭配低成本 workhorse model

Aaron Levie 认为 multi-model agentic systems 会成为未来核心模式。他引用 Cursor 的研究称，用 frontier model 负责 planning 和 orchestration，再把明确指令交给更便宜的 workhorse model 执行，可显著降低项目总 token 成本，带来 15X cost improvement。关键在于复杂任务中只有分解、设计决策和权衡等少数时刻需要 frontier intelligence，其他大量执行步骤可由低成本模型完成。应用层的差异化将来自领域理解和 model routing 能力。
