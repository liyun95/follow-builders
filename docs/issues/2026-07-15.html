Aaron Levie · ceo @box

Aaron Levie：agent 普及需要把知识工作也做成可测试、可 eval 的系统

Aaron Levie 指出 code 适合 agents 的关键原因之一是可以快速测试：手动验证应用或运行 test。但大多数企业工作只有在真实世界交付后才能知道结果。他认为未来会出现一整套测试其他工作的机会，也意味着更多 agents 会进入 workflows；企业需要更好的 evals 来判断 model、prompt 或 system 改动是否让知识工作变好或变坏。
