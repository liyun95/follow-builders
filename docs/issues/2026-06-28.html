Training Data · Podcast

Engram 谈 memory 与 continual learning：让模型把团队 context 学进权重

本期采访 Engram 的 Dan Biderman 和 Jessy Lin，讨论 memory 与 continual learning。Engram 的核心观点是，模型瓶颈不只是 raw intelligence，而是理解新的、持续变化的 context；相比只把记忆放进 context window 或外部数据库，他们希望用 adapter fine tuning、LoRA、supervised fine tuning、RL、on-policy distillation 等方法，把团队 workspace 中重要的 context 持续训练进模型权重。访谈还提到，与 Notion、Microsoft、Harvey 等 workspace 场景合作时，目标是训练 per team models，让模型像在公司工作多年的人一样理解项目、流程、优先级和工具设置，并可能带来显著 token inference consumption 降低。
